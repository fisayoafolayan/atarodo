# Atarodo

A minimal, content-focused Ghost theme inspired by clean design and great reading experiences.

**[Live Demo](https://fisayoafolayan.com/blog)**

## Features

- Clean, text-focused post feed
- Full-width cover image hero on blog posts with gradient overlay
- Dark mode with system preference detection and manual toggle
- Reading progress bar on posts
- Code syntax highlighting with copy button (highlight.js)
- Social sharing (X, Facebook, LinkedIn, Email)
- Responsive navigation with mobile hamburger menu
- Ghost native search integration
- Ghost native comments support
- Back to top button
- Smooth anchor scrolling with TOC highlighting
- Previous/next post navigation
- Tag and author archive pages
- Static page support
- Custom 404 error page

## Installation

### From zip

1. Download the latest release zip
2. Go to your Ghost Admin > Settings > Design > Change theme
3. Upload the zip file
4. Activate Atarodo

### For development

```bash
git clone https://github.com/fisayoafolayan/atarodo.git
cd atarodo
npm install

# Start Ghost with Docker
npm run dev

# Visit http://localhost:2368
# Admin: http://localhost:2368/ghost/
```

The theme directory is mounted directly into Ghost. Edit source files (`assets/css/style.css` and `assets/js/script.js`), then run `npm run build` to generate the minified versions. Changes to `.hbs` files are picked up on browser refresh. Restart the container after modifying `package.json`.

## Build commands

| Command | What it does |
|---------|-------------|
| `npm run build` | Minifies CSS and JS |
| `npm run css` | Minifies CSS only |
| `npm run js` | Minifies JS only |
| `npm run zip` | Lints, builds, and creates `atarodo.zip` for upload |
| `npm run dev` | Starts Ghost via Docker |
| `npm run lint` | Runs gscan theme validator |

### Build tools

- [lightningcss](https://lightningcss.dev/) - CSS minification and vendor prefixing
- [terser](https://terser.org/) - JS minification

Source files are `assets/css/style.css` and `assets/js/script.js`. The minified output (`*.min.css`, `*.min.js`) is what Ghost loads.

## Configuration

### Navigation

Set up your navigation links in Ghost Admin > Settings > Navigation.

### Logo

Upload a logo in Ghost Admin > Settings > General > Publication icon & logo. If no logo is set, only navigation links appear in the nav bar.

### Dark mode

The theme toggle is available in the nav bar and footer. It cycles through System > Light > Dark. The preference is saved in localStorage.

### Comments

Atarodo supports Ghost's native comments. Enable them in Ghost Admin > Settings > Membership > Commenting.

For third-party comment systems like Remark42, use Ghost's Code Injection (Settings > Code injection > Site footer). The theme provides a `.post-comments` container in the post template that injected scripts can target.

### Search

Search is powered by Ghost's built-in Sodo Search. The search icon in the nav bar triggers it automatically.

## Theme structure

```
atarodo/
  assets/
    css/
      style.css           - Source styles (CSS custom properties)
      style.min.css       - Minified output (generated by build)
    js/
      script.js           - Source JS (vanilla, no dependencies)
      script.min.js       - Minified output (generated by build)
  partials/
    loop.hbs              - Post card for the feed
    pagination.hbs        - Pagination with arrows
  default.hbs             - Base layout
  index.hbs               - Homepage / post feed
  post.hbs                - Blog post
  page.hbs                - Static page
  tag.hbs                 - Tag archive
  author.hbs              - Author archive
  error.hbs               - Error page (404, 500)
  package.json            - Theme metadata, config, and build scripts
```

## CSS custom properties

The theme uses CSS custom properties for easy customization:

```css
--color-bg              /* Page background */
--color-text            /* Primary text */
--color-text-secondary  /* Secondary/muted text */
--color-border          /* Borders and dividers */
--color-card-bg         /* Card backgrounds */
--color-code-bg         /* Code block backgrounds */
--color-accent          /* Accent color (inherits from Ghost Admin) */
--font-body             /* Body font stack */
--font-heading          /* Heading font stack */
--font-mono             /* Monospace font stack */
--content-width         /* Max width of post content */
--wide-width            /* Max width of outer containers */
```

## Ghost compatibility

- Requires Ghost >= 6.0.0
- Supports `{{@page.show_title_and_feature_image}}` for pages
- Supports `{{comments}}` native comments
- Supports `data-ghost-search` for Sodo Search
- Supports Ghost's accent color via `--ghost-accent-color`
